En una terminal:

$> sudo docker run --rm -it -p 6080:6080 -p 6081:6081 -p 6082:6082 -p 6083:6083 -p 8080:6084  turtlebot2

En otra distina ejecutar:

$> ./launch.sh

Dentro de launch lo que se ejecuta es lo siguiente:

$> sudo docker ps

Del resultado, copiamos lo que ponga en el campo "CONTAINER ID" y lo pegamos en otras 2 o 3 terminales:

$> sudo docker exec -it [CONTAINER_ID] /bin/bash

Dentro de la terminal, escribimos lo siguiente:
